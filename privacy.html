Freeze — Privacy Policy

Last updated: April 8, 2026

Overview

Freeze is a focus timer browser extension. It works together with the Freeze companion desktop app running on your own machine. This policy describes what data the extension handles and how.

Data We Collect

  • Email address and authentication token — if you create an account via the companion app, your email and a session token are stored locally in the extension (chrome.storage.local) to keep you signed in.
  • Timer session records — when a session ends, the extension saves the start time, end time, duration, mode (stopwatch/countdown/Pomodoro), and whether it was completed. This data is stored locally and, if you are signed in, sent to the companion app running on your own machine (http://127.0.0.1:5555).
  • Preferences — your chosen notification sound and timer presets are stored locally in the extension.

How Data Is Used

All collected data is used solely to operate the timer and display your session history. No data is used for advertising, analytics, or any purpose unrelated to the extension's timer functionality.

Data Sharing

We do not sell, share, or transfer your data to any third party. All communication occurs between the extension and the Freeze companion app on your own local machine. No data is sent to external servers.

Data Storage

Data is stored in your browser's local extension storage (chrome.storage.local) and optionally in the Freeze companion app's local database on your machine. You can clear extension data at any time by removing the extension from Chrome.

Permissions

The extension requests the following permissions, each used only for the timer's core function:

  • storage — save timer state and preferences locally.
  • alarms — trigger phase-end events and badge updates accurately in the background.
  • notifications — alert you when a focus block or break ends.
  • sidePanel — display the timer UI in the browser side panel.
  • offscreen — keep the toolbar badge countdown updated every second.

The content script runs on all pages solely to play a notification sound when a timer phase ends. It does not read, collect, or modify any web page content.

Contact

If you have questions about this policy, please open an issue on the project repository.
